# Memulai Proyek

1. Clone repository ini:

   ```bash
   git clone https://github.com/marcellandreas/Belajar-Cepat-Vue.js-Project-e-commerce
   ```

2. Masuk ke folder proyek:

   ```bash
   cd Belajar-Cepat-Vue.js-Project-e-commerce
   ```

3. Instal dependencies:

   ```bash
   npm install
   ```

4. Jalankan aplikasi:

   ```bash
   npm run serve
   ```

5. Buka di browser:
   ```bash
   http://localhost:8080
   ```
